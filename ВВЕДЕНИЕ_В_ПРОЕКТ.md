# ВВЕДЕНИЕ В ПРОЕКТ

## Тема проекта
**MakarovFlow** — веб-приложение для управления личной продуктивностью и мониторинга психоэмоционального состояния в формате Telegram Mini App.

---

## 1. Актуальность проекта

В современном мире студенты и молодые специалисты сталкиваются с высокими академическими и профессиональными требованиями, что часто приводит к стрессу, выгоранию и снижению продуктивности. Согласно исследованиям, более 60% студентов испытывают трудности с организацией времени и отслеживанием своего эмоционального состояния.

Существующие решения для трекинга настроения и управления задачами либо слишком сложны для повседневного использования, либо не обеспечивают комплексного подхода к анализу личных данных. Кроме того, большинство таких приложений требуют регистрации, хранят данные на внешних серверах и не интегрированы в привычные пользователям коммуникационные платформы.

MakarovFlow решает эти проблемы, предоставляя:
- Простой и интуитивный интерфейс внутри Telegram
- Полную приватность данных (локальное хранение)
- Комплексный подход: настроение + задачи + учёба
- AI-аналитику паттернов поведения
- Кроссплатформенность без необходимости установки

---

## 2. Цель проекта

**Основная цель** — разработать интегрированное веб-приложение для Telegram, которое помогает пользователям систематически отслеживать своё эмоциональное состояние, управлять задачами и академическими обязательствами, получая при этом персонализированную аналитику для повышения личной эффективности.

**Практическая цель** — создать работающий прототип, доступный в виде Telegram Mini App, который может использоваться для ежедневного мониторинга состояния и планирования задач без необходимости регистрации или передачи данных третьим лицам.

## 7. Структура проекта

Проект состоит из следующих ключевых компонентов:

### 7.1. Frontend (Telegram Mini App)
**Технологии**: React 18, Vite, TailwindCSS, Framer Motion

**Основные модули**:
- **Journal (Дневник)** — трекинг настроения, энергии, сна с календарём и аналитикой
- **Tasks (Задачи)** — менеджер задач в стиле Things 3 с фильтрами и напоминаниями
- **AI Assistant (AI-помощник)** — интеллектуальный анализ данных и рекомендации
- **Settings (Настройки)** — управление языком, экспорт данных, информация

### 7.2. Backend (Telegram Bot)
**Технологии**: Node.js, Grammy (Telegram Bot framework)

**Функции**:
- Обработка команды `/start`
- Предоставление Web App URL
- Keep-alive механизм для предотвращения засыпания (Render free tier)

### 7.3. База данных
**Технология**: IndexedDB (через Dexie.js)

**Схема**:
- `journal_entries` — записи дневника (настроение, энергия, сон, заметки)
- `tasks` — задачи с датами, напоминаниями, статусами
- `schedule` — расписание занятий
- `homework` — домашние задания
- `ai_messages` — история сообщений с AI
- `settings` — настройки пользователя
- `user_stats` — статистика использования

### 7.4. Дополнительные системы
- **i18next** — система локализации (русский/английский)
- **Zustand** — управление глобальным состоянием
- **Framer Motion** — система анимаций
- **Telegram Mini App SDK** — интеграция с Telegram (haptic feedback, theme)

## 10. Ограничения и допущения

### 10.1. Технические ограничения
- Зависимость от Telegram платформы
- Ограничения IndexedDB (хранилище браузера)
- Отсутствие синхронизации между устройствами
- Ограничения free tier Render.com (засыпание после 15 мин. неактивности)

### 10.2. Функциональные допущения
- AI-аналитика основана на keyword matching, а не на реальных ML-моделях
- Уведомления не реализованы (требуется интеграция с Telegram Bot API)
- Backup данных выполняется вручную через экспорт JSON

### 10.3. Пользовательские ограничения
- Требуется аккаунт Telegram
- Необходим современный браузер с поддержкой IndexedDB
- Данные не синхронизируются между устройствами

## 12. Заключение

MakarovFlow представляет собой современное решение для мониторинга психоэмоционального состояния и управления личной продуктивностью. Проект объединяет актуальные веб-технологии, принципы user-centered design и подход privacy-first, создавая удобный инструмент для ежедневного использования.

Реализация проекта в формате Telegram Mini App обеспечивает мгновенный доступ для миллионов пользователей без необходимости установки дополнительных приложений. Локальное хранение данных гарантирует полную приватность, что особенно важно при работе с персональной информацией о настроении и здоровье.

Проект имеет значительный потенциал для дальнейшего развития и может служить основой для более масштабных исследований в области цифрового self-tracking и персонализированной аналитики продуктивности.
